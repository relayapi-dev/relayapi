## Official SDKs

### Node.js / TypeScript

```bash
npm install @relayapi/sdk
```

```typescript
import Relay from '@relayapi/sdk';

const client = new Relay(); // reads RELAY_API_KEY from environment
```

[GitHub](https://github.com/relayapi-dev/relay-node) · [npm](https://www.npmjs.com/package/@relayapi/sdk)

### Python

```bash
pip install relay
```

```python
from relay import Relay

client = Relay()  # reads RELAY_API_KEY from environment
```

[GitHub](https://github.com/relayapi-dev/relay-python) · [PyPI](https://pypi.org/project/relay/)

### Go

```bash
go get github.com/relayapi-dev/relay-go
```

```go
import relaygo "github.com/relayapi-dev/relay-go"

client := relaygo.NewClient() // reads RELAY_API_KEY from environment
```

[GitHub](https://github.com/relayapi-dev/relay-go)

