Use RelayAPI directly from your favorite AI coding assistant. Each integration gives the AI full access to the RelayAPI — creating posts, managing accounts, uploading media, and more — without leaving your editor or terminal.

| Integration | Platform | Install method |
|---|---|---|
| [Claude Code](/resources/integrations/claude) | Claude Code CLI & Desktop | Plugin marketplace |
| [Cursor](/resources/integrations/cursor) | Cursor IDE | Plugin marketplace |
| [OpenAI Codex](/resources/integrations/codex) | OpenAI Codex CLI | Repository plugin |
| [OpenClaw](/resources/integrations/openclaw) | OpenClaw AI assistant | ClawHub skill |

All integrations support the full RelayAPI feature set across 21 platforms and authenticate via your existing API key.

## Getting an API Key

Every integration requires a RelayAPI API key. If you don't have one:

1. Sign up at [relayapi.dev](https://relayapi.dev/signup)
2. Go to **Settings > API Keys**
3. Create a new key — it starts with `rlay_live_` (production) or `rlay_test_` (testing)

See the [Quickstart](/quickstart) for a full walkthrough.
