## March 2026

### Initial Release

- Launch of the RelayAPI unified social media API
- Support for 17 platforms: Twitter/X, Instagram, Facebook, LinkedIn, TikTok, YouTube, Pinterest, Reddit, Bluesky, Threads, Telegram, Snapchat, Discord, Mastodon, WhatsApp, Google Business Profile, SMS
- Post creation, scheduling, and publishing
- Media upload with R2 storage
- Webhook notifications
- API key authentication
- OpenAPI 3.1.0 specification
- Interactive API documentation
